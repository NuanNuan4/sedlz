# 第6章 - 数据可视化交互式网页

基于Jupyter Notebook《第6章 坐标轴与图表设计》制作的交互式数据可视化教程网页，提供了丰富的图表展示和实时交互功能。

## 📋 项目概述

这个项目将传统的matplotlib数据可视化教程转换为现代化的网页应用，保留了所有原始数据分析逻辑，同时提供了更好的用户体验和交互功能。网页包含了5个主要章节，涵盖了坐标轴的各个方面和高级绘图技术。

## ✨ 功能特性

### 📊 完整的教程内容
- **6.1 坐标轴概述** - 理解坐标轴的基本结构和组成
- **6.2 向任意位置添加坐标轴** - 动态调整坐标轴位置和大小
- **6.3 日期时间坐标轴** - 时间序列数据的可视化处理
- **6.4 坐标轴脊的设置** - 自定义图表边框样式
- **6.5 三角函数图表** - 数学函数的交互式展示

### 🎨 现代化界面设计
- 响应式布局，适配各种设备屏幕
- 渐变色彩主题，现代化视觉效果
- 卡片式设计，清晰的内容组织
- 平滑动画过渡，提升用户体验

### ⚡ 丰富的交互功能
- **实时参数调整** - 滑块、复选框等控件即时更新图表
- **预设样式** - 一键应用常用图表样式
- **函数计算器** - 实时计算三角函数值
- **数据导出** - 将图表数据导出为JSON格式
- **重置功能** - 快速恢复默认设置

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 无需安装额外软件，直接在浏览器中运行

### 启动方法
1. 下载项目文件到本地
2. 直接用浏览器打开 `index.html` 文件
3. 或者通过HTTP服务器运行：
   ```bash
   # 使用Python内置服务器
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   
   # 使用Node.js的http-server
   npx http-server
   ```

## 📖 使用指南

### 导航使用
- 左侧边栏提供章节导航
- 点击章节标题快速跳转到对应内容
- 移动端可通过顶部菜单按钮打开/关闭侧边栏

### 交互控件说明

#### 6.1 坐标轴概述
- **显示轴脊复选框** - 控制坐标轴边框的显示
- **更新图表按钮** - 重新生成随机数据点

#### 6.2 位置控制
- **X/Y位置滑块** - 调整坐标轴在画布中的位置
- **大小滑块** - 控制坐标轴的尺寸

#### 6.3 日期时间坐标轴
- **时间间隔选择** - 设置刻度的时间间隔
- **日期格式选择** - 选择时间标签的显示格式
- **刻度旋转角度** - 调整时间标签的倾斜角度
- **刻度标签大小** - 控制文字大小

#### 6.4 坐标轴脊设置
- **预设样式按钮** - 快速应用传统、最小化、中心化样式
- **自定义设置** - 独立控制每个轴脊的显示、颜色和宽度
- **多边形/五边形图表** - 展示不同的图形样式

#### 6.5 三角函数图表
- **函数显示复选框** - 选择显示正弦或余弦函数
- **X轴范围选择** - 调整函数的显示范围
- **数据点数量** - 控制图表的平滑程度
- **动画效果** - 开启/关闭图表动画
- **函数值计算器** - 输入x值计算对应的三角函数值

### 快速操作
- **导出数据** - 将所有图表数据导出为JSON文件
- **重置图表** - 一键恢复所有设置到默认状态
- **主题切换** - 在明亮/暗色主题间切换（功能预留）
